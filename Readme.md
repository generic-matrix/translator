# How to build the Android App

* ```git clone https://github.com/generic-matrix/translator.git```
* cd react-native-app
* npm install
* react-native run-android


# How To build iOS App

* ```git clone https://github.com/generic-matrix/translator.git```
* cd react-native-app
* npm install
* react-native run-ios
